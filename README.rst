=============================
django-shells
=============================

Better shells for your `manage.py`.

Documentation
-------------

The full documentation is at https://django-shells.readthedocs.org.

Quickstart
----------

Install django-shells::

    pip install django-shells

Then add ``'shells'`` to your ``INSTALLED_APPS``.
